# 스포츠 유학 일정 생성기

## 변경 반영 내용
- 날짜는 **달력에서 클릭한 날짜만** 결과에 표시됩니다.
- 종목 필터를 제거하고 **리그 필터만** 남겼습니다.
- 지도는 **Google Maps** 기준으로 구성했습니다.
- 경기 클릭 시 해당 지역으로 이동하고, 기본은 **같은 지역의 야구장/축구장 전체가 보이도록** 맞춥니다.
- 지도 아래 체크박스 `해당 경기장만 보기` 체크 시 선택한 구장만 남깁니다.
- 검색 결과는 **날짜 헤더를 크게** 보이도록 바꿨고, 경기 카드에는 **⚾ / ⚽ 이모지**를 넣었습니다.
- 지역 텍스트는 결과 목록에서 숨겼습니다.
- "축구장·야구장이 가까운 지역" 섹션은 제거했습니다.

## 실행
```bash
npm install
cp .env.example .env.local
# .env.local 에 Google Maps JavaScript API 키 입력
npm run dev
```

## 데이터 구조
- `data/games.ts`: 샘플 경기 데이터
- `data/leagues.ts`: 리그별 표시 이미지와 공식 사이트

## 로고 관련
- 리그 이미지는 공식 사이트 호스팅 URL을 사용하도록 넣었습니다.
- 팀 로고는 현재 가독성 우선으로 텍스트 배지 fallback 구조입니다.
- 팀별 공식 엠블럼을 붙이려면 `data/types.ts` 의 `Team.logoUrl` 필드와 렌더링 부분을 연결하면 됩니다.
